# App Icons

Diese Datei erklärt, wie Sie die benötigten App-Icons generieren.

## Benötigte Icon-Größen

Die PWA benötigt folgende Icon-Größen:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Icon generieren

### Option 1: Online Icon Generator (empfohlen)

1. Besuchen Sie: https://www.pwabuilder.com/imageGenerator
2. Laden Sie ein quadratisches Bild hoch (mindestens 512x512 px)
3. Wählen Sie "Generate"
4. Laden Sie das ZIP herunter
5. Entpacken Sie die Icons in diesen Ordner

### Option 2: Figma oder Adobe XD

Erstellen Sie ein Icon mit:
- Hintergrundfarbe: #1a1a2e (dunkelblau)
- Primärfarbe: #3b82f6 (blau)
- Akzentfarbe: #f59e0b (orange)
- Symbol: Spanische Flagge, Buch oder "ES" Buchstaben

Exportieren Sie in allen benötigten Größen.

### Option 3: Canva

1. Erstellen Sie ein Design (512x512)
2. Verwenden Sie das Farbschema der App
3. Exportieren Sie als PNG
4. Nutzen Sie einen Icon-Generator für die anderen Größen

## Dateinamen

Benennen Sie die Icons wie folgt:
- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Design-Empfehlung

Für ein einfaches Icon können Sie:
- Einen dunkelblauen Hintergrund (#1a1a2e) verwenden
- Ein weißes oder blaues "ES" in der Mitte platzieren
- Optional: Ein kleines Spanien-Flagge-Element hinzufügen

## Temporäre Lösung

Bis Sie eigene Icons haben, können Sie das Projekt auch ohne Icons verwenden.
Die PWA-Installation wird trotzdem funktionieren, nur das Icon wird dann das
Browser-Standard-Icon sein.
